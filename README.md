### 📊 Quasi-Causal Analysis of Dispatch Delays & Late-Delivery Risk

<a href="https://github.com/bhavishjain7133/quasi-causal-delivery-delay-analysis">
  <img src="https://github-readme-stats.vercel.app/api/pin/?username=bhavishjain7133&repo=quasi-causal-delivery-delay-analysis&theme=nord&border_color=7dd3fc&title_color=0EA5E9&icon_color=7dd3fc" alt="causal delivery repo" />
</a>

Cross-fitted, doubly-robust **(AIPW)** causal study of how missing a seller's dispatch deadline affects late-delivery risk across **81,941 Olist orders**, with a preregistered-style protocol and falsification tests.

| Layer | Technology |
| :--- | :--- |
| Language | Python |
| Method | Causal Inference, 5-fold Cross-fitted AIPW |
| Data | Olist relational e-commerce dataset |
| Rigor | Reproducible notebooks, automated tests, GitHub Actions CI |
