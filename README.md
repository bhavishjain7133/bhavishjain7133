### Interpretable Learning-Augmented Last-Mile Delivery Routing

<a href="https://github.com/bhavishjain7133/learning-augmented-last-mile-routing">
  <img src="https://github-readme-stats.vercel.app/api/pin/?username=bhavishjain7133&repo=learning-augmented-last-mile-routing&theme=nord&title_color=38BDF8&text_color=38BDF8&icon_color=38BDF8&border_color=38BDF8" alt="Interpretable learning-augmented last-mile routing repository card" />
</a>

Developed an interpretable hierarchical zone-preference model using 6,112 Amazon last-mile delivery routes. The model reduced the aggregate median official route score by **39.9%** against nearest-neighbour routing on 926 unseen test routes and reduced median zone re-entries from 21 to 0.

### Quasi-Causal Analysis of Dispatch Delays and Late-Delivery Risk

<a href="https://github.com/bhavishjain7133/quasi-causal-delivery-delay-analysis">
  <img src="https://github-readme-stats.vercel.app/api/pin/?username=bhavishjain7133&repo=quasi-causal-delivery-delay-analysis&theme=nord&title_color=38BDF8&text_color=38BDF8&icon_color=38BDF8&border_color=38BDF8" alt="Quasi-causal delivery delay analysis repository card" />
</a>

Conducted a quasi-causal supply-chain study on 81,941 Olist orders. Using five-fold cross-fitted AIPW, the analysis estimated a **14.11 percentage-point** higher adjusted late-delivery risk following dispatch-deadline breaches, supported by overlap, balance, robustness, temporal, cohort, and placebo checks.

| Layer | Technology |
|---|---|
| Programming | Python |
| Research | Causal Inference, Supply Chain Analytics |
| Estimation | Augmented Inverse Probability Weighting (AIPW) |
| Data | Relational E-Commerce Order Data |
| Validation | Cross-Fitting, Overlap and Covariate-Balance Diagnostics |
| Engineering | Reproducible Notebooks, Automated Tests, GitHub Actions CI |
